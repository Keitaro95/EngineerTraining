# PDF OCR Skill

このスキルは次の処理を行います:

1. 指定された PDF を読み込む
2. 内部の画像・文字列を認識する (OCR)
3. テキストを構造化して出力

## 手順
1. ユーザーに PDF のパスを聞く
2. Python ライブラリ (pdfplumber または pytesseract/pdf2image) で OCR
3. 結果をテキスト出力

## 使用ツール
- Python スクリプト (実行可能)

## その他
- スライド内の図（アーキテクチャ図やフローチャート）は、Mermaid形式のコードに変換してください
